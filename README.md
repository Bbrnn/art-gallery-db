# Art Gallery Inventory & Sales Database

## Description
This project provides a relational database schema for managing an art galleryâ€™s inventory, exhibitions, sales, artists, and buyers. The system is designed using MySQL and supports tracking artworks, organizing exhibitions, recording sales, and managing artist and buyer information.

## Features
- Manage artists and their artworks
- Track artwork status (available/sold)
- Organize exhibitions and associate artworks with exhibitions
- Record sales and buyer information
